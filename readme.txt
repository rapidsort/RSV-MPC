== Description ==
RSV MPC is plugin for Create Multiple Pages in single click.

== Installation ==
1. Upload the <strong>rsv-mpc</strong> folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the '<strong>Plugins</strong>' menu in WordPress
3. Go to RSV MPC in your WordPress admin, add text as multiple lines in textarea and click on Create Pages Button.
4. RSV MPC will take each line as new page and it will create pages according to your text.
